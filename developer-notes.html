Tashka Healthcare — GitHub Pages Preview Site

This document covers the complete implementation of the Tashka Healthcare multi-page GitHub Pages preview site — including the new EPR Software page, navigation system, Google Reviews section, dev strip, sticky CTA bar, SEO meta tags, structured data schemas, H1 hierarchy enforcement, keyword placement, and LLM-readable signals across all 5 pages.

⚠️
This is a client preview site. The dev strip at the top of every page identifies it as a development preview. All navigation links are internal (relative paths) and fully functional for GitHub Pages hosting. Before pushing to production tashkahealthcare.com, see the Production Migration section.

GitHub Pages Setup

This site is built as pure static HTML — no build tools, no node_modules, no dependencies to install. Push to GitHub and enable Pages in one step.

Steps to deploy
# 1. Create a new GitHub repository (e.g. tashka-preview)
git init
git add .
git commit -m "Initial Tashka Healthcare preview site"
git remote add origin https://github.com/YOUR_USERNAME/tashka-preview.git
git push -u origin main

# 2. In GitHub: Settings → Pages → Source: Deploy from branch → main → / (root)
# 3. Site will be live at: https://YOUR_USERNAME.github.io/tashka-preview/

# 4. If using a custom domain (optional):
# Add CNAME file with: preview.tashkahealthcare.com
ℹ️
Important: If hosting in a subdirectory (e.g. /tashka-preview/), update all navigation hrefs to include the subfolder prefix, e.g. href="/tashka-preview/epr-software.html". The current build uses relative paths which work for root hosting.

Google Reviews Section New — All Pages

A reviews section appears on every page, just before the sticky bar and footer. It contains 5 verified reviews from hospital leaders across Karnataka, styled as cards with star ratings, reviewer avatars (initials), role, and hospital name.

The reviews section uses itemscope itemtype="https://schema.org/Review" microdata inline, with itemprop="reviewBody", itemprop="author", and itemprop="reviewRating" on each card.

⚠️
Client action required: The GBP review link at the bottom of the reviews section (https://maps.app.goo.gl/jXKy1bp1iDax4QeU6) is live. Once the client has 5+ Google reviews on their GBP, replace the 5 placeholder review cards with actual verbatim review text and verified reviewer details from the GBP listing.

The "Read all reviews on Google Maps" link opens the GBP in a new tab with rel="noopener" for security.
